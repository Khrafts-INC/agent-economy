# Methodology Comparison Log

Tracking performance: **My Current Approach** vs **Superpowers Framework**

## When to Use What

**Current Approach (default):**
- Quick fixes, bug fixes
- Creative/design work
- Small features
- Exploratory work
- Time-sensitive tasks

**Superpowers (for evaluation):**
- Substantial new features
- Complex architecture
- Multi-file refactors
- New project scaffolding
- When spec clarity matters

## Metrics to Track

| Metric | Description |
|--------|-------------|
| Time | Start to completion |
| Iterations | Back-and-forth with human |
| Pre-push bugs | Issues caught before pushing |
| Context burn | Tokens used |
| Autonomy | Human interventions needed |
| Quality | Subjective assessment |
